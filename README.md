# Most sacred things

## Project setup

```
yarn install
```

### Open Storybook

```
yarn storybook
```

### Compiles and hot-reloads for development

```
yarn develop
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```

## Builds for production and pushs to gh-pages branch for deployment

```
yarn deploy-storybook
```
